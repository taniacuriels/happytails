# Messaging Package

Conversation threads, message events, and moderation rules.

## Features

- Conversation management
- Message threading
- Moderation and escalation
- Audit trails
- Attachment handling

## Interfaces

- `ConversationService` - Manage conversations
- `MessageService` - Send and retrieve messages
- `ModerationService` - Content moderation

## Usage

```typescript
import { ConversationService } from '@happytails/messaging';
```
