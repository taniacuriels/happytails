# Scheduling Package

Timeline management, appointments, and conflict detection.

## Features

- Timeline events
- Appointment scheduling
- Conflict detection
- Bulk rescheduling
- Calendar overlays

## Interfaces

- `TimelineService` - Create and retrieve events
- `AppointmentService` - Manage appointments
- `ConflictService` - Detect and resolve conflicts

## Usage

```typescript
import { TimelineService } from '@happytails/scheduling';
```
