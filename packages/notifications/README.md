# Notifications Package

Notification triggers, rules, and preference management.

## Features

- Push notifications
- Email notifications
- Trigger system
- Preference management
- Quiet hours support

## Interfaces

- `NotificationService` - Send notifications
- `PreferenceService` - Manage user preferences
- `TriggerService` - Register notification triggers

## Usage

```typescript
import { NotificationService } from '@happytails/notifications';
```
