# Web App

HappyTails web platform for shelter administrators, coordinators, and power users.

## Features

- Dashboard overview
- Animal management
- Foster assignment coordination
- Bulk operations
- Message moderation
- Timeline management
- Data reporting

## Tech Stack

- **Next.js** - React framework
- **TypeScript** - Type safety
- **React Query** - Data fetching
- **Tailwind CSS** - Styling

## Development

```bash
npm run dev
```

Runs on http://localhost:3000

## Building

```bash
npm run build
npm run start
```
